This is a very first introduction to Reinforcement Learning, which mainly refers to UCL RL course given by David Silver and the book Reinforcement Learning An Introduction by Sutton.

Here are some useful resources: https://github.com/jiahaoliu1891/Reinforcement-Learning-101

Introduction to Basic Ideas

Reinforcement Learning is based on the following model, which shows the interaction between Agent and Environment :

where Agent take (orsay output) Action \(A_t\) depending on the State \(S_t\) at time step \(t\). The Environment decides the next State \(S_{t+1}\) given the current State \(S_t\) and Action \(A_t\) . Meanwhile, it send the Agent a Reward \(R_t\).

The Core Question for Reinforcement Learning is : " How should the Agent react to the Environment? " Formally, this is to ask the Agent to learn an optimal ==Policy \(π_*(a|s)\).==

To answer this question, we divide it to a few smaller problems:

Q1: How to mathematically model the interaction between Agent and Environment ? A1: Markov Processes → Markov Reward Processes → Markov Decision Processes (MDP)

Q2: How to solve Markov Decision Processes, , i.e., finding the optimal ==Policy \(π_*(a|s)\)== ? Q2.1 Predition Problem: Given a Policy \(π\), how to evaluate each State \(S\) and each Action \(A\)? Formally, how to compute the value \(v_π(s) \in R\) for State \(s\), where \(v_π(s)\) is a real number that shows how good is a State \(S\) ? Similarly, \(q(s,a)\) for Action \(a\). A2.1: Bellman Expectation Equations (Can be directly solved) Q2.2 Control Problem: Then, what's the optimal ==Policy \(π_*(a|s)\)== , state-value function \(v_*(s)\) and state-action function \(q_*(s,a)\)? A2.2: Bellman Optimality Equations (Can not be directly solved)

Q3: Bellman Optimality Equations has NO trivial solutions, how to solve it efficiently? Q3.1: Suppose we know the MDP: A3.1: Dynamic Programming (DP) method. Q3.2: Suppose we do not know the MDP: A3.2: Monte Carlo (MC) and Temporal Difference (TD) method.
